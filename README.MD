# YouTube to RSS

Getting the RSS link to a youtube video is anoying. This script helps to extract the ID from a YouTube Channel or Playlist, and the RSS link.

**How to use the Script**
You need python3 installed on your computer. Then just enter:

```bash
python3 get_id.py
Enter YouTube URL: https://https://www.youtube.com/@PewDiePie/videos #Enter the URL to the YouTube channel or playlist here
#the output should be
Channel ID: UC-lHJZR3Gqxm24_Vd_AJ5Yw
RSS link: https://www.youtube.com/feeds/videos.xml?channel_id=UC-lHJZR3Gqxm24_Vd_AJ5Yw
```
